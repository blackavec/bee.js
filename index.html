Modules

There are many modules which you can use by calling it by wizz() method. also for those who are a little old fashion, we provided some alternation such as:

  • bee.start(module);
  • bee.load(module);
<script type="text/javascript">
   bee.wizz(module);
</script>

honeycomb honeycomb is a module that handle the ajax requests. honeycomb has some chain methods which you can make a query with those and then execute it by calling spread() method.

bee.wizz('honeycomb')
   .request('http://domain.com/index.html')
   .method('get')
   .on('success', function (result) {
      // result 
      console.log('success', result);
    })
   .resultData('text')
   .spread();
